# student-performance-analysis
Data analysis using python ( Pandas, Matplotlib)
##  Objective
Analyze student data to understand factors affecting performance.

##  Tools Used
- Python
- Pandas
- Matplotlib

##  Analysis Performed
- Data cleaning
- Missing value check
- Duplicate removal
- Average grade calculation
- Correlation analysis
- Visualization (Histogram, Scatter Plot, Bar Chart)

## Key Insights
- Study time has a positive impact on grades
- Gender-based performance comparison done
- Most students scored in mid-range

This project demonstrates basic data analysis and visualization skills using Python.
